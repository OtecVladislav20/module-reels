Модуль с функционалом риллсов

Обязательные библиотеки:

- swiper - используется для свайпа элементов
- vg-modals - используется для открытия модальных окон, в нашем случае открытия видео

Функционал:

- Открытие модального окна с историями на нужном слайде
- Остановка видео при зажатии слайда
- Промотка видео по нажати. на скролл бар
- Поддержка фото и видео
- Загрузка нескольких фото и видео в одну историю
